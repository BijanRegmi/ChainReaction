# Chain Reaction

Sasta copy of the original "Chain Reaction" game.

## Building

1. Clone the repository
```bash
git clone https://github.com/BijanRegmi/ChainReaction
```
2. Install dependencies

[SFML](https://www.sfml-dev.org/download.php)

3. Compile and run

```bash
mkdir bin && make run
```
